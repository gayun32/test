# 📚 나만의 프롬프트 관리 프로그램

콘솔에서 실행하는 **프롬프트 관리 도구**입니다.
프롬프트를 카테고리별로 등록하고, 검색하고, 즐겨찾기로 관리할 수 있습니다.

---

## 🚀 실행 방법

```bash
python prompt_manager.py
```

---

## ✨ 기능 목록

| 번호 | 기능 | 설명 |
|:---:|---|---|
| 1 | 프롬프트 추가 | 제목, 내용, 카테고리를 입력해 새 프롬프트를 등록 |
| 2 | 전체 목록 보기 | 등록된 모든 프롬프트를 번호와 함께 출력 |
| 3 | 카테고리별 조회 | 선택한 카테고리의 프롬프트만 조회 |
| 4 | 키워드 검색 | 제목/내용에 포함된 키워드로 검색 |
| 5 | 상세 보기 | 프롬프트 번호로 전체 내용 확인 |
| 6 | 즐겨찾기 추가/해제 | ⭐ 즐겨찾기 상태를 토글 |
| 7 | 즐겨찾기 목록 보기 | 즐겨찾기한 프롬프트만 모아보기 |

---

## 🗂️ 카테고리 종류

- 📝 텍스트 생성
- 🖼️ 이미지 생성
- 🎬 영상 생성
- 🎭 페르소나
- 🤖 자동화
- 📦 기타

---

## 💡 참고

- 프로그램 실행 중에 추가한 프롬프트와 즐겨찾기 상태는 **실행 중에만 유지**되며, 종료 시 초기화됩니다.
- 기본 프롬프트 3개가 미리 등록되어 있습니다.
